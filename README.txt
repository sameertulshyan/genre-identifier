## Genre Identifier



## Build Instructions
This application has been tested on Java 7 and 8.

In order to run this program:

1. Compile the .java files
2. Run the IdentifyCorrectGenre object file and provide as arguments:
	1. A valid path to the JSON file containing the information for each book and its description
	2. A valid path to the CSV file containing the information for each genre and its keywords

In addition to the JRE System library, this application references the following libraries and will require them to run correctly:

1. JSON Simple (https://code.google.com/archive/p/json-simple/downloads)
2. JUnit (not required to use the application, only the testing classes) (https://github.com/junit-team/junit5)
